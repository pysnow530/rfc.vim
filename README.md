## rfc.vim

A vim rfc syntax plugin.

## Installation

Use your favorite plugin manager.

Using [Vundle](https://github.com/VundleVim/Vundle.vim):

```vim
Plugin 'pysnow530/rfc.vim'
```

## Quick start guide

Use rfc folding:

```vim
let g:rfc_folding = 1
```
